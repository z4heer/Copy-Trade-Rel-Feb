Trading Application User Manual

User Setup

1. Create My App at NuvamaWealth API-Connect

Set up your account at NuvamaWealth by using their API-Connect platform.

2. TOTP Setup

Configure Time-based One-Time Password (TOTP) for enhanced security.

3. Generate Request ID / Session ID

Generate a unique Request ID or Session ID for API interactions.

Redirect the user to the login url - https://www.nuvamawealth.com/api-connect/login?api_key=04FCaiaW37pOKA

4. Configure users.xlsx File

Create and configure an Excel file named users.xlsx with the following columns:

  • userid: User ID
  • api_key: API Key
  • api_secret_password: API Secret Password
  • reqId: Request ID

5. Trading Operation Pre-requisite

Ensure all users in users.xlsx are configured with the required parameters and an active Request ID.

Activities

1. List Users

Purpose: Display a list of active users with a fresh Request ID.

Usage: Use this feature to verify active users before placing orders.

2. Buy Order

Purpose: Place a buy order for all active users at once.

Procedure:

  1. Initiate the buy order process for all active users.
  2. After completion, you will be redirected to the "List Orders" section.

3. List Orders

Purpose: Display a list of existing orders.

Features:

  • Links to modify or cancel orders.

4. List Net Positions

Purpose: Display a list of net positions.

Features:

  • Links to sell orders.

5. List Holdings

Purpose: Display a list of holdings.

Features:

  • Links to sell orders.

6. Modify Order

Purpose: Modify a single order.

Procedure: Select the order you wish to modify and update the necessary details.

7. Cancel Order

Purpose: Cancel a single order.

Procedure: Select the order you wish to cancel and confirm the cancellation.

8. Sell Order

Purpose: Sell a single order.

Procedure: Select the order you wish to sell and confirm the sale.

Ensure all the pre-requisites are met and configured correctly in the users.xlsx file before performing any trading operations. This will help in smooth and error-free execution of trading activities.

Flow Progress

User Setup -> List Users -> Buy Order -> List Orders -> List Net Positions -> List Holdings -> Modify Order -> Cancel Order -> Sell Order
